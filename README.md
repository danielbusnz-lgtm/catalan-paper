# On the Greatest Common Divisor of Consecutive Catalan Numbers

**Daniel Brooks** — University of Chicago

## Abstract

We investigate the greatest common divisor of consecutive Catalan numbers C(n) and C(n+1), and establish conditions under which consecutive Catalan numbers share a common factor.

## Results

**Theorem 1:** C(n) divides C(n+1) if and only if n = 0, 1, or 4.

**Theorem 2:** If p = n + 2 is prime and p > 5, then GCD(C(n), C(n+1)) = C(n) / p.
